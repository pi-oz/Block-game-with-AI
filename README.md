This is an AI-based rule-driven game where the player competes against an intelligent AI opponent. The objective is simple but challenging: survive for as long as possible while avoiding continuously falling blocks that appear at random positions on the screen.

As the game progresses, blocks keep dropping infinitely, making the challenge increasingly difficult and testing the reflexes and decision-making abilities of both the player and the AI.

Key Features:
🤖 AI Opponent based on predefined game rules.
🎮 Player vs AI Competition in real-time.
🧱 Infinite Random Falling Obstacles for endless gameplay.
📈 Dual Scoreboards to track performance:
AI Loss Counter – Displays the total number of times the AI loses.
Player Loss Counter – Displays the total number of times the player loses.
